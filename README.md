# Crossover Service

## Development setup

#### - Install NodeJS
#### - Install tools

- `sudo npm install -g firebase-tools`
(more info: https://firebase.google.com/docs/cli/)

optional:
- `sudo npm install -g nodemon`

#### - Install dependencies

- `npm install`

#### - Run Service
by default, just type:
`node ./service.js`

 or,
`nodemon ./service.js`

This will start the local development service.
